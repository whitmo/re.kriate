Extended Page Toggle (double-tap)

Three page buttons have extended pages. Tap once for the primary page; tap the same button again to toggle the extended page.

trigger ↔ ratchet (button x=6)
note ↔ alt note (button x=7)
octave ↔ glide (button x=8)

Duration and velocity have no extended pages — pressing them again stays on the same page.

Loop Editing

Hold the loop button (x=12 on row 8), then press two columns in the step area to set the loop start and end for the current page/track.

Example: set loop from step 3 to step 10

Brighter cells are within the loop. The loop applies to the current page's parameter on the active track. Each parameter has its own independent loop — this is the core of polymetric sequencing.

Direction Modes

Each track has a direction mode controlling how all its parameters advance through their loops.

Polymetric Sequencing

The key concept: each parameter has its own independent loop length. When different parameters have different loop lengths, they cycle at different rates, creating evolving patterns.

Example

A single track with different loop lengths per parameter:

trigger
loop: 1-16
note
loop: 1-8
octave
loop: 1-5
velocity
loop: 1-3

After 16 steps, trigger has cycled once, note twice, octave 3+ times, and velocity 5+ times — the combination doesn't repeat for 240 steps (LCM of 16, 8, 5, 3).
