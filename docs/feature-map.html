Core Engine
Sequencer

lib/sequencer.lua (232 lines)

Clock-driven per-track coroutines. Step advancement with per-parameter independent loops. Swing/shuffle 0-100%. Ratchet repeats. Probability note drops. Portamento/glide. Sprite visual voice output alongside audio.

Complete
Track Data Model

lib/track.lua (208 lines)

9 parameters: trigger, note, octave, duration, velocity, probability (core) + ratchet, alt_note, glide (extended). 4 tracks × 16 steps. Per-param clock divider. Musically useful defaults.

Complete
Direction

lib/direction.lua (94 lines)

5 playhead modes: forward, reverse, pendulum (bounce), drunk (random walk), random. Pendulum tracks direction state internally.

Complete
Scale

lib/scale.lua (37 lines)

14 scales via musicutil (Major through Chromatic). 8 octaves from configurable root. Scale degree + octave → MIDI note conversion.

Complete

10 Pages

Note, Octave, Duration, Velocity, Probability — 7-row bar graphs
Ratchet, Alt Note, Glide — Extended pages (double-tap nav to toggle)

Grid Provider

lib/grid_provider.lua

Pluggable backend: monome (hardware), midigrid (MIDI emulation), simulated (seamstress visual). Registration + factory pattern.

Complete
Sprite Renderer

lib/seamstress/sprite_render.lua

7 shapes (circle, rect, triangle, diamond, star, line, dot). Track colors, echo trails, glide lines, beat grid, playhead. Drift + float movement.

Complete
Keyboard

lib/seamstress/keyboard.lua

Full keybinding set: space=play, r=reset, 1-4=track, qwety=pages, d=direction, ctrl+s/l=save/load, ctrl+1-9=pattern slots.

Complete
Voice Backends
MIDI

lib/voices/midi.lua (66 lines)

Note on/off, per-channel routing, portamento CC 5/65, all-notes-off CC 123. Monophonic with cancel-previous.

Complete
OSC

lib/voices/osc.lua (49 lines)

Configurable host/port. Paths: /rekriate/track/{n}/note, note_off, all_notes_off, portamento.

Complete
Sprite / Visual

lib/voices/sprite.lua (168 lines)

Additive visual voice alongside audio. 7 shapes, RGBA track colors, velocity→size, octave→Y, alt_note→X. Echo + glide rendering.

Complete

Wrapper around norns nb player. Defers to nb param system for voice selection and pitch control.

Complete

Pattern Storage
In-Memory Patterns

lib/pattern.lua (44 lines)

16 slots for instant save/recall during performance. Deep-copy save preserves full track state. Grid UI shows slot occupancy on rows 1-2.

Complete
Disk Persistence

lib/pattern_persistence.lua

.krp files with Adler-32 checksum validation. Platform-aware paths: norns ~/dust/data/, seamstress XDG. Save/load/list/delete operations. Sanitized filenames.

Complete
Platform Support
norns

re_kriate.lua (72 lines)

Thin global hooks (init/redraw/key/enc/cleanup) delegating to lib/app. nb voice integration. Screen metro at 15fps. Hardware grid via monome provider.

Complete
seamstress

seamstress.lua (93 lines)

Grid simulation on screen with mouse clicks. Sprite rendering. Keyboard input. MIDI port 1. Screen metro at 30fps. Full dev environment without hardware.
