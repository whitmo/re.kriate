Kria Grid Layout

re.kriate feature map -- original kria vs current implementation
Trigger Page
Rows 1-4 show all 4 tracks simultaneously. Each column = one step (1-16). Lit cells = active triggers. IMPLEMENTED -- This page works correctly in re.kriate.
Note Page
Rows 1-7 display scale degrees for the active track. Row 1 = degree 7 (highest), Row 7 = degree 1 (lowest). Each column = one step. The active value shows as a lit cell; cells below show a dim bar. IMPLEMENTED
Octave Page
Original: Row 1 has 6 keys for per-track base octave (+0 to +5). Rows 2-7 show per-step octave values. re.kriate: Uses generic value page layout (rows 1-7 all for per-step values). PARTIAL -- Missing base octave selector on row 1.
Duration Page
Original: Row 1 = duration multiplier selector (1x-8x). Rows 2-7 = downward sliders (lower = longer). re.kriate: Uses generic value page (rows 1-7 all for per-step values). PARTIAL -- Missing duration multiplier on row 1.
Velocity Page (re.kriate Extension)
This page does NOT exist in original kria (which outputs CV/gate, not MIDI). Per-step velocity is a re.kriate addition for MIDI output. Rows 1-7 = velocity values. Same bar visualization as other value pages. EXTENSION
Ratchet/Repeat Page (Extended Trigger)
Accessed by pressing the Trigger key a SECOND TIME (double-press toggle). The Trigger key blinks to indicate extended mode. Rows 2-6 = sub-trigger pattern (up to 5 subdivisions per step). Row 1 = increase subdivisions. Row 7 = decrease subdivisions. NOT IMPLEMENTED
Alt Note Page (Extended Note)
Accessed by pressing the Note key a SECOND TIME. Same layout as Note page (rows 1-7 = scale degrees). ADDITIVE: values from both Note and Alt Note pages combine. Result = (note + alt_note) mod scale_length. Has its own independent loop, creating polymetric note combinations. NOT IMPLEMENTED
Glide Page (Extended Octave)
Accessed by pressing the Octave key a SECOND TIME. Controls portamento/slew per step. Row 1 = 120ms (max glide), Row 7 = 0ms (no glide). Each row adds ~20ms of slew time. NOT IMPLEMENTED
Loop Modifier (Hold)
Hold the Loop key on row 8. Press two columns to set loop start/end for the active parameter. Original: col 11. re.kriate: col 12. Also: hold Loop + press track button = toggle mute. PARTIAL -- Loop editing works. Missing: wrapping loops (end before start), mute-via-loop gesture.
Time/Division Modifier (Hold)
Hold the Time key on row 8. Row 1 shows 16 selectable clock divisions (1-16) for the CURRENT PARAMETER. Each parameter can have its own clock division, enabling per-parameter polymetric timing. NOT IMPLEMENTED -- re.kriate only has per-track division via params menu.
Probability Modifier (Hold)
Hold the Prob key on row 8. Rows 2-5 show per-step probability as vertical faders. Row 2 = 100% (always fire), Row 3 = ~75%, Row 4 = ~50%, Row 5 = ~25%. Each parameter has its own probability. NOT IMPLEMENTED
Scale Page
Left (cols 1-8): Per-track config. Col 1 = TT clock, Col 2 = Trigger clock, Cols 4-8 = Direction modes (Forward / Reverse / Triangle / Drunk / Random) for each of the 4 tracks (rows 1-4). Right (cols 9-16): Scale interval editor. Rows 6-7: Scale preset slots. NOT IMPLEMENTED -- Scale only editable via params menu. No direction modes on grid.
Pattern Page
Row 1: 16 pattern slots. Momentary press = load pattern. Hold = store current pattern. Hold Pattern key + press pattern = cue for next loop boundary. Row 2: cue clock divider. NOT IMPLEMENTED -- Col 16 is play/stop in re.kriate, not pattern access.
Meta-Sequencer
Activated by holding Pattern key + pressing row 7. Sequences patterns over time. Row 1: pattern select for current meta-step. Row 2: meta-step clock divider. Rows 3-6: 64-step meta-sequence (4 rows x 16 cols). Row 7: meta-sequence duration per step. NOT IMPLEMENTED

Per-Parameter Independent Loops

Each parameter (trigger, note, octave, duration, etc.) has its own loop start and end point. A trigger loop of 8 steps combined with a note loop of 5 steps creates evolving polymetric patterns that only repeat after 40 steps (LCM of 8 and 5).

TRIGGER (8 steps)
NOTE (5 steps)

Extended Pages (Double-Press Toggle)

Three parameter keys have extended/secondary pages accessed by pressing the same key twice:

Trigger -> Ratchet (sub-trigger patterns)
Note -> Alt Note (additive second layer)
Octave -> Glide (portamento per step)

The page key blinks when in extended mode. Press again to return to the primary page.

Direction Modes (Scale Page)

Each track can independently use one of 5 playback directions. Configured on the Scale page (cols 4-8, rows 1-4):

Forward -> (default, implemented)
Reverse <- (not implemented)
Triangle <-> (bounce, not impl)
Drunk ~> (random walk, not impl)
Random ?? (any step, not impl)
